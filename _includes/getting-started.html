尝试 Dotty

如果您是 Mac 用户，您可以使用 brew:

brew install lampepfl/brew/dotty

如果您是 Linux 用户，请在本页面下载 Dotty 最新发布版。您可以将 Dotty 存放目录下的 bin/ 子目录加入系统 PATH 环境变量中。

现在您可以编译 Scala 源代码了：

dotc hello.scala

您也可以使用 dotr 命令启动 Dotty repl。

您也可以使用 Scastie 在浏览器中试用 Dotty。

创建 Dotty 项目

创建一个新的 Dotty 项目最快捷的方式是使用 sbt (1.1.4+)。

创建一个 Dotty 项目：

sbt new lampepfl/dotty.g8

或者创建一个 Dotty 与 Scala 2 混合编译项目：

sbt new lampepfl/dotty-cross.g8

请参见 Dotty Example Projec 查阅相关文档。
